# Rhythm Paradise Groove Wiki

Independent guide site for Rhythm Paradise Groove at `https://www.rhythmparadisegroove.wiki`.

The site is built on the existing Next.js / MkSaaS stack and focuses on:

- Rhythm Paradise Groove / Rhythm Heaven Groove regional name coverage
- Perfect routes, minigame cue notes, walkthrough structure, and future per-minigame expansion
- TV calibration, input lag, demo save transfer, and Switch setup guidance
- Beatspell, multiplayer, Drum Lessons, Rhythm Toy Box, Cafe, and mode coverage
- Source-checked official Nintendo links, trailer references, and guide pages

## Development

```bash
pnpm install
pnpm dev
pnpm lint
pnpm build
```

## Content Rules

- Do not add fake codes, tier lists, character pages, or RPG-style databases that the game does not support.
- Treat `Rhythm Paradise Groove` as the canonical site keyword and `Rhythm Heaven Groove` as the North American alias.
- Prefer official Nintendo pages for platform, release, demo, and feature facts.
- Use YouTube videos as source references, not transcript material.
- Expand per-minigame pages only when search demand and source confidence justify them.
- Keep the site clearly labeled as an independent guide hub.
